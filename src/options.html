Help

  • Container: Select which container the rule applies to. "No Container" means default browsing.
  • Allow: URL can open in this container (non-exclusive).
  • Allow Only: URL MUST open ONLY in this container (exclusive).
  • URL Pattern: Regex pattern to match URLs (e.g., "mail.google.com", ".*\\.google\\.com").
  • High Priority: Rules with high priority take precedence over others.
  • Invalid Rule: "Allow Only" + "*" pattern is forbidden as it blocks all navigation.
